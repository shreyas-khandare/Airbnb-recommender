# Airbnb Recommender System

## Project Overview
This is a Python-based Airbnb recommendation system that predicts and suggests the best Airbnb listings to users based on their preferences. 
It uses a Random Forest Regression model trained on multiple listing features such as city, room type, bedrooms, bathrooms, accommodates, and more. 
A Tkinter GUI allows users to input their preferences and get the top recommendations instantly.

## Features
- Predicts Airbnb listing prices based on user inputs.
- Filters listings according to preferences: city, room type, bedrooms, bathrooms, accommodates, and maximum price.
- Displays top 5 recommended listings with predicted prices.
- GUI built with Tkinter for a simple and interactive interface.
